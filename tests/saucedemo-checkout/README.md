# Tests — SauceDemo Checkout

This directory contains automated Playwright test scripts for the SCRUM-101 checkout workflow.

## Structure
- `checkout-happy-path.spec.ts` — Happy path end-to-end checkout (generated by playwright-test-generator)
- `checkout-validation.spec.ts` — Form validation and error handling tests (generated by playwright-test-generator)
- `checkout-navigation.spec.ts` — Navigation flow and back-button behavior (generated by playwright-test-generator)

> Test scripts in this directory are generated and healed by the `playwright-test-generator` and `playwright-test-healer` agents.
